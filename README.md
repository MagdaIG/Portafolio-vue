# Portafolio vue

Portafolio personal desarrollado con Vue.js que muestra mis habilidades como desarrolladora web y UX Designer. Incluye secciones de proyectos, experiencia, habilidades técnicas y contacto. Diseño responsivo y moderno con gradientes y animaciones.

## Tech Stack

Vue.js, JavaScript and CSS3

## Getting Started

```bash
git clone https://github.com/MagdaIG/Portafolio-vue.git
cd Portafolio-vue
npm install
npm run dev
```
